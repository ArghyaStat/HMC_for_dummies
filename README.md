 # HMC_for_dummies

 ## Simulations folder
 
 Contains HMC_for_dummies.R for simulation studies from Sections 2-4 in the article.

 Contains the figures for Sections 2-4 in the article. 

 ## Bayesian_logistic_regression folder
 
 Contains example_implementation.R for the application of HMC on the Pima Indian diabetes dataset from Section 5 in the article.

 Contains figures for Section 5 in the article.

 Contains posterior_beta_warmup.RData and posterior_beta_samples.RData of warm-up HMC chain and pre-conditioned HMC chain, respectively, for Section 5.
